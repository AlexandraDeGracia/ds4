(1 pt) ¿Qué diferencia hay entre git pull y git fetch?
fetch Descargar objetos y referencias desde otro repositorio
pull Descargar e integrar cambios desde otro repositorio o rama local

te manda  error por que no te descargara el cambios para el repositorio
(1 pt) ¿Qué comando usas para combinar ramas?
checkout para pasar a una rama a otra
